# 🚀 3D Dark Mode E-Commerce Web Application

A **premium, futuristic e-commerce platform** with a **3D-inspired dark mode UI**, smooth animations, and a modern user experience.
Designed to feel like a **next-generation shopping application** with immersive visuals and high performance.

---

## 🌌 Features

### 👤 User Features

* User authentication (Login / Signup)
* Browse products with search & filters
* Interactive product detail view
* Add to cart & manage cart
* Smooth checkout process
* Order history & tracking

---

### 🛠️ Admin Features

* Admin dashboard
* Add / edit / delete products
* Manage orders and update status
* User management

---

### 🎨 UI/UX Highlights

* 🌑 Dark mode with neon accents
* ✨ Glassmorphism design (blur + transparency)
* 🌌 3D-inspired interface with depth and layering
* ⚡ Smooth animations & micro-interactions
* 📱 Fully responsive (mobile, tablet, desktop)

---

### ⚡ Advanced Features

* Wishlist system ❤️
* Product ratings & reviews ⭐
* Smart search and filtering
* Toast notifications
* Loading skeletons
* Optimized performance

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

```bash
npm install
```

### 3. Configure environment variables

Create a `.env` file and add:

```
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
```

### 4. Run the application

```bash
npm run dev
```
